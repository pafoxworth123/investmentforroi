## 🏠 About
Free real estate investment calculator tool for BRRRR analysis, Cash-on-Cash returns, DSCR calculations, and Fix-and-Flip profitability. Built for small investors, house hackers, and landlords.

## 🚀 Features
- **BRRRR Calculator** - Buy, Rehab, Rent, Refinance, Repeat analysis
- **Cash-on-Cash Return** - Quick ROI for rental properties
- **DSCR** - Debt Service Coverage Ratio for lenders
- **Fix & Flip** - Profit projections for flippers
- Mobile-optimized, zero dependencies

## 💻 Tech Stack
